# GrafosC-Asm
## Proyecto final del curso de compiladores

### Este proyecto busca implementar un compilador que traduzca un lenguaje tipo C a un lenguaje ensamblador para la arquitectura MIPS. En base a diagramas de flujo construidos en MERMAID.

### El proyecto se divide en los siguientes apartados:
1. Interfaz grafica tipo Drag n Drop para la contruccion de diagramas de flujo (Mermaid) y exportacion  (Aun no implementado)
2. Analizador de Mermaid y traductor de C 
3. Compilador de C a Ensamblador
4. Generador de código MIPS (Aun no implementado)
